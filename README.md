
## 🚀 KaaryaUp - Team CodeSlayers - AkhilaDas & SanikaKangane

**Hustle. Earn. Level Up.**

KaaryaUp is a freelancing and skill-development platform designed for Indian youth aged 11–24. It transforms traditional jobs into engaging “quests,” helping users earn money, build skills, and stay motivated through game-like progression.

---

## 🌟 Problem

* Freelancing platforms feel complex and boring for beginners
* Young people lack engaging ways to earn and learn
* No system combines **earning + skill growth + gamification**
* Users often lose motivation quickly

---

## 💡 Solution

KaaryaUp turns freelancing into a **game-like experience**:

* Jobs become **Quests**
* Users earn **XP (Experience Points)**
* Level up with progress
* Unlock **badges and achievements**
* AI assistant **YuvaAI** guides users

---

## 🎯 Key Features

### 🧩 Quest-Based Freelancing

* Micro-tasks and gigs presented as “quests”
* Categories: Design, Writing, Coding, Marketing, etc.

### 🎮 Gamification System

* XP points for every completed task
* Level progression system
* Daily streaks and challenges
* Badges and rewards

### 🤖 YuvaAI Assistant

* Suggests tasks based on skills
* Helps users improve
* Provides learning guidance

### ⚡ Fast & Simple Onboarding

* Minimal signup details:

  * Name
  * Age
  * Skills
  * Email (Gmail verification)
  * Mobile number (OTP verification)

### 📈 Skill Growth

* Learn while earning
* Track progress and performance
* Build a strong portfolio

---

## 👥 Target Audience

* Students (11–24 years)
* Beginners in freelancing
* Creators & skill-based earners
* Youth who prefer independence over traditional jobs

---

## 🛠️ Tech Stack (Suggested)

* **Frontend:** HTML, CSS, JavaScript / React
* **Backend:** Firebase / Node.js
* **Database:** Firestore / MongoDB
* **Authentication:** Google Sign-In + OTP
* **AI Integration:** Google AI Studio (YuvaAI)

---

## 🎮 How It Works

1. User signs up quickly
2. Selects skills
3. Gets recommended quests
4. Completes tasks
5. Earns XP + money
6. Levels up and unlocks rewards

---

## 🔥 Unique Value

Unlike traditional freelancing platforms, KaaryaUp focuses on:

* Engagement through gamification
* Youth-friendly UI/UX
* Motivation through rewards & levels
* Learning + earning combined

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
